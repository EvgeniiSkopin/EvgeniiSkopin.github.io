Task

Develop an interface of the event application for holding internal conferences of employees of the Pyaterochka retail chain

Role

UX/UI Designer

Company

NameOne

At the first stage...

At the first stage of work, I had only a list of functions required for this application. Each of these functions needed to be visualized in order to better understand the operation of the application. For this purpose, I've made rough sketches on paper, it allowed me to structure the interface and continue its detailed design.

Easy scanning of topics

The app design included the comprehensive and diverse functionality which had to be presented to users in a simple and clear way. The designers had to analyze and prioritize all the points, as there was a high risk of overloading the screen. By research and analysis, the user scenarios were created to determine which information about the meal in the recipe is found the most important.

Learn about the lecturer

For the user it is possible to view the profile of the lecturer. Profile is presented in a separate page, which provides information about the lecturer, his location and list of lectures held by them. By the end of the lecture, the user has the opportunity to evaluate the lecture and ask a question.

The style of the elements fully inherits the main style of Pyaterochka.

Manage the application

In addition to the design of the application, the task was to develop an administrative part through which managers could control the application. An administrative part is a desktop application. The style of the interface is as simple and clear as possible, the colors used are neutral. You can view all the application screens and click here.
