Task

App is used for patients on company insurance plans to manage their care.

Role

UI Designer

Company

NameOne

Project task

I worked on this project only as a UI designer. The entire structure of the application was presented to me by the customer; these were the schemes for each screen that it was forbidden to deviate from. My task was to design a visually attractive interface using elements of corporate identity.

Main screen

The main function of the application is to provide timely medical care for insured persons. Home page is appointments. Here, the user can select someone from the family or himself as a subject of medical care, view a list with the nearest appointments and add a new one.
